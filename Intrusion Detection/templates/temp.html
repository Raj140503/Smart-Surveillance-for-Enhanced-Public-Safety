Database Overview

Welcome to the Intrusion Detection Database. This section provides an overview of all the individuals currently registered in the system for face recognition. Each registered person has a unique ID, name, and a set of captured facial images used to train the recognition model.

The primary purpose of this database is to distinguish between authorized and unauthorized individuals in real-time. When the detection system identifies a face, it compares it with this database:

  • ✅ If matched, the person is labeled as authorized.
  • ❌ If not matched, the system flags them as unauthorized and triggers an alert if conditions are met (e.g., detection duration).
